FAQ - Perguntas Frequentes

1. O que é HTML?
HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar a estrutura básica de uma página web.
2. Como adicionar texto a uma página HTML?
Você pode usar a tag <p> para parágrafos ou as tags de cabeçalho <h1>, <h2>, etc., para títulos.
